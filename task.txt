Цель:
Создать демонстрационное приложение для проверки работы ключевых возможностей Livewire 4, FluxUI и Blaze compiler в рамках TALL stack.

Требования

1. Использовать:

   * Laravel (последняя версия)
   * Livewire 4
   * FluxUI
   * Blaze compiler

2. Реализовать компонент с:

   * реактивным состоянием
   * вычисляемыми свойствами (computed)
   * обработкой событий (event dispatch)
   * использованием wire:intersect

Функциональность

1. Список элементов

* Отобразить список карточек (например, посты или продукты)
* Данные можно захардкодить или взять из базы

2. Lazy loading через intersect

* Загружать дополнительные элементы при появлении блока в viewport
* Использовать wire:intersect

3. Computed свойства

* Создать computed property:

  * например: фильтрация или подсчет элементов
* Отображать результат в UI

4. События (Event Dispatch)

* Реализовать:

  * dispatch события при клике на элемент
  * слушатель события в другом компоненте
* Пример:

  * клик по карточке → событие → отображение деталей

5. Интеграция FluxUI

* Использовать компоненты FluxUI:

  * кнопки
  * карточки
  * модальные окна

Ожидаемый результат

* Полностью рабочий Livewire-компонент
* Демонстрация:

  * intersect
  * dispatch событий
  * computed свойств
* Чистый и понятный UI с использованием FluxUI
* Минимум JS (упор на Livewire)

* Реализовать пагинацию
